Класс ActivatableMovingObjectController

Класс предназначен для управления движением активируемого объекта. Объект может перемещаться вертикально или горизонтально между двумя позициями и изменять своё состояние при активации или деактивации. Подробнее...

Базовые классы:MonoBehaviour и IActivateble.
